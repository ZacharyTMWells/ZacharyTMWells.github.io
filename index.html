Stage 3: 3D Printing

Stage 3 is the process of 3D printing the mechanism and the arm sections

FDM 3D printing is the process of melting a filament and then extruding it onto a print bead in layers to then make up a solid object

The Use of a standard FDM or Fused deposition modelling 3D printer allowed me to take the CAD file and turn it into reality. To make 3D printing easier when making the CAD file i made it so the majority of components where flat allowing for multiple shorter easier prints, Though the downside to this is that some assembly is required after printing.

Above is the mechanism after 3D printing

Due to some issues with the 3D printer used the layers of the larger prints from the side don't look as good as hoped this could be down to poor extrusion or too much play in the printers frame causing it to unexpectedly shift.
This could be remedied by better maintenance of the printer or the use of a different printer.
Another issue present is the stringing that has occurred, this is an issue with the retraction settings of the printer and can be fixed with proper calibration

Additional Technologies

In addition to the technologies used above, others where considered for the project some that would be used in future developments and others that wouldn’t suit the project

Resin Printing

Resin 3D printing was looked at however it was deemed unsuitable for this project as the items created by a resin printer though high in detail are limited by the build volume of the printer

Silicone Casting

Another technology looked at was silicone and resin casting. By using silicone we would have be able to add a synthetic skin to the arm, this idea however would have to be scrapped due to time constraints and delays causing the use of silicone to become unfeasible, it would however be something that i would like to have a look at attempting in the future.

Haptic Pen

During the modeling stage of the project we could have used the haptic pen to edit the 3D scanned model, however due to the model needing to be decimated there was no point in adding extra detail plus the model was already in an acceptable state alreadyDuring the modelling stage of the project we could have used the haptic pen to edit the 3D scanned model, however due to the model needing to be decimated there was no point in adding extra detail plus the model was already in an acceptable state already
